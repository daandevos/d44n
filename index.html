Zo ben ik mezelf

Over mij

Programmeren is al een lange tijd een dagelijkse bezigheid. Ik hou van de vrijheid die ik als programmeur heb om te ontwikkelen wat ik maar wil.

Sinds 2017 heb ik mijn eigen bedrijf. Daar ben ik erg blij mee, het doel van mijn bedrijf is het realiseren van mooie, overzichtelijke en vooral nuttige websites voor kleine tot middelgrote bedrijven.

Dit doe ik dagelijks

Mijn hobby's

Dagelijks ben ik bezig met mijn ontwikkeling. In het weekend werk ik aan projectjes. Zowel voor mijn bedrijf als persoonlijk, ik hou ervan om met nieuwe technieken aan de gang te gaan.

Wanneer ik rust zoek kijk ik een serie op Netflix, spreek ik af met vrienden of lees ik artikelen op Twitter, NU of Tweakers.

Passie voor programmeren

Zo begon ik ooit

Vanaf mijn 13e leeftijd begon ik met programmeren. Dit kwam omdat ik met een oude vriend een online spel ging spelen, we verveelden ons in de zomervakantie. Uit interesse besloot ik mij te verdiepen in de manier waarop zo'n spel gemaakt wordt om mijn eigen variant ervan te maken. Na een aantal weken code copy+pasten lukte het. Ik vond het gaaf om iets te laten werken door code te typen. Door de ambitie heb ik mij ontwikkeld als Backend Developer en leer ik nog dagelijks.
